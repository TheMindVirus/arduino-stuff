# Intel Active Management Technology Emulator
# IOCTL Boilerplate for Testing Motherboard Chipset Web Servers
# Powered by Intel Management Engine and AMD DASH on PCH via DMI, IPMI or iDRAC

A lightweight Arduino version of the commands found on \
PC Motherboards with Intel and AMD Chipsets supporting \
on-board programmable web servers accessible across \
a Live Firmware Engine / BIOS Management Interface
